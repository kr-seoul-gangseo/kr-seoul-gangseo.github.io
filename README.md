# 강서구 · 서울특별시

> 강서구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-seoul-gangseo.github.io/**

강서구는 김포국제공항이라는 하늘길과 마곡의 첨단 산업단지를 함께 품은 서울 서부의 관문 자치구입니다. 조선시대 양천현(陽川縣)이 자리했던 옛 고을로, 진경산수화의 대가 겸재 정선이 현령으로 부임해 한강 절경을 화폭에 담은 예술의 무대이기도 합니다. 오늘날에는 LG사이언스파크와 코오롱, 넥센 등 대기업 연구소가 몰린 마곡지구가 조성되면서 항공·바이오·화학 R&D의 거점으로 변모했습니다. 아시아 최대 규모의 서울식물원과 허준의 자취를 간직한 가양동 일대가 어우러져 산업과 문화, 자연이 공존하는 도시로 발전하고 있습니다. 송파구에 이어 서울에서 두 번째로 인구가 많은 자치구로, 화곡동 주거지역과 마곡 신도심이 대비를 이루며 성장하고 있습니다.
